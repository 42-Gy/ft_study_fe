# Week1 과제: SUBAK 마이페이지 (Vanilla JS)

## 목표

- Mock API를 연결해 데이터를 화면에 렌더링한다.
- 로딩/성공/실패 상태를 구분해 사용자에게 표시한다.
- 탭/필터/정렬 같은 기본 인터랙션을 구현한다.

## 제공 파일

- `index.html` / `style.css`: 기본 UI
- `mockApi.js`: `MockApi.getUsersMe` 제공
- `app.js`: 구현해야 할 스켈레톤

## 필수 요구사항

- 내 정보(이름/이메일/코인/역할 등) 렌더링
- 대여 현황(사물함/구역/대여 시작/만료/연체) 렌더링
- 아이템 기록 테이블 렌더링
- 코인 기록 테이블 렌더링
- 로딩/성공/실패 상태 표시
- 토큰 미입력 시 401 에러 처리

## 선택 요구사항

- 탭 UI 접근성(`role`, `aria-selected`) 보완
- 아이템 상태 필터/정렬 로직 구현
- hover/transition 1개 이상 추가
- 빈 상태(데이터 0개) UI 구성

## 구현 힌트

- `MockApi.getUsersMe({ token })` 호출 후 `state.me`에 저장
- `Bearer ` 접두가 없는 토큰은 자동으로 붙이기
- 날짜 정렬은 `Date`로 변환 후 비교
